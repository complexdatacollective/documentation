Развертывание собственного экземпляра Fresco

1. Настройка необходимых служб

  • Создайте базу данных с PlanetScale
  • Создайте новое приложение на uploadthing

2. Используйте кнопку развертывания Vercel для настройки развертывания вашего проекта в Vercel.

Примечание: Для безопасности у вас есть пять минут с момента развертывания приложения для создания учетной записи пользователя. Если этот срок истекает без создания учетной записи пользователя, ваша конфигурация устареет. Вы можете повторно развернуть, следуя тем же шагам.

3. Создание репозитория Git

Следуйте инструкциям по созданию репозитория Git для развертывания. В нем будет клонированная версия этого репозитория.

4. Настройка проекта

Укажите необходимые переменные среды из служб, которые вы настроили на шаге 1.

ПеременнаяОписание
DATABASE_URLСтрока подключения к базе данных
UPLOADTHING_SECRETКлюч API для вашего приложения uploadthing
UPLOADTHING_APP_IDИдентификатор приложения для вашего приложения uploadthing

5. Развертывание

Нажмите "Развернуть" и дождитесь завершения развертывания.

6. Создание учетной записи пользователя

Посетите ваше развернутое приложение, чтобы создать учетную запись вашего администратора. Может быть создана только одна учетная запись пользователя.
